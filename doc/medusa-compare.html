Here's a quick comparison of Hydra and Medusa. This is based on my understanding of Hydra 5.0 and Medusa 1.5. It should be noted that even if a particular item is checked, it does not necessarily mean it works correctly. I have not used Hydra in a number of years. If newer versions have addressed some of the known issues or added new functionality that should be reflected here, please let me know.
